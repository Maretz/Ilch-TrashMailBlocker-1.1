TrashMail Blocker für Ilch 1.1p

Mit dieser Erweiterung können TrashMail Nutzer bei der Registrierung ausgeschlossen werden.
Im Adminbereich in der Konfiguration kann die Blacklist bearbeitet werden.

Installation
____________________________________

Vorsicht! Es wird die include/contents/user/regist.php ersetzt! Ggf. vorher sichern.


Den Inhalt aus dem Ordner upload nach include uploaden.
Im Anschluss die installation aufrufen ( http://www.meine-seite.de/index.php?installation ) und Anweisungen folgen.
